A lightweight Unreal Engine project demonstrating a soft-body “jelly bullet” effect using dynamic non-uniform scaling, deformation curves, impact-normal–based squash/stretch, and physics-driven secondary motion.
这是一个使用虚幻引擎（Unreal Engine）的轻量级项目，展示了通过动态非均匀缩放、变形曲线、基于碰撞法线的挤压/拉伸以及物理驱动的次级运动来实现软体“果冻子弹”效果。
